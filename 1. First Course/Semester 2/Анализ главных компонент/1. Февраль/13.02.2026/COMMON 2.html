PCA — Практические занятия 1–4

PCA — Principal Component Analysis (анализ главных компонент).

Фокус: от первых ручных реализаций до визуализации и кластеризации.

Инструменты: Python, NumPy, scikit‑learn, Google Colab, matplotlib/plotly.
Примеры: PCA на Iris (sklearn), PCA в NumPy & sklearn (Kaggle).

Структура практик 1–4

  • Практика 1: Разведочный анализ (EDA), выбор датасетов, постановка задач.
  • Практика 2: Реализация PCA «с нуля» в NumPy + сравнение с scikit‑learn (документация PCA).
  • Практика 3: Explained variance, scree‑plot, biplot, интерпретация компонент (пример Iris, PCA & biplot).
  • Практика 4: PCA для визуализации и кластеризации (k‑means на PC‑пространстве; пример: Kaggle: Clustering & PCA).

Напоминание терминов

  • PCA — Principal Component Analysis (анализ главных компонент).
  • PC — Principal Component (главная компонента, вектор в пространстве признаков).
  • Scores — координаты объектов в пространстве главных компонент.
  • Loadings — веса признаков в компонентах (координаты признаков в векторе компоненты).
  • Explained variance — доля дисперсии, объяснённая компонентой.
  • Scree‑plot — график собственных значений / долей объяснённой дисперсии.
  • Biplot — совмещение scores и loadings на одном графике.

Практика 2: математика PCA

  • Данные: \(X \in \mathbb{R}^{n \times p}\), строки — объекты, столбцы — признаки.
  • Центрированные данные: \[ \tilde{X}_{ij} = X_{ij} - \bar{X}_{\cdot j}, \] где \(\bar{X}_{\cdot j}\) — среднее по признаку \(j\).
  • Ковариационная матрица: \[ S = \frac{1}{n-1}\tilde{X}^\top \tilde{X}. \]
  • Собственные значения/векторы: \[ S w_k = \lambda_k w_k,\ \lambda_1 \ge \dots \ge \lambda_p. \]
  • Scores (координаты объектов): \(Z = \tilde{X} W\), где \(W = [w_1,\dots,w_p].\)

Практика 2: NumPy PCA (Iris)


import numpy as np
from sklearn.datasets import load_iris

iris = load_iris()
X = iris["data"]
y = iris["target"]

# 1. Центрирование
Xc = X - X.mean(axis=0, keepdims=True)

# 2. Ковариационная матрица
S = Xc.T @ Xc / (Xc.shape[0] - 1)

# 3. Собственные значения/векторы
eigvals, eigvecs = np.linalg.eigh(S)
idx = np.argsort(eigvals)[::-1]
eigvals = eigvals[idx]
eigvecs = eigvecs[:, idx]

# 4. Scores (проекции) на первые 2 компоненты
Z = Xc @ eigvecs[:, :2]

Задание. Рассмотрите точки в \(\mathbb{R}^2\):

  • \(x_1 = (1, 2)\), \(x_2 = (2, 3)\), \(x_3 = (3, 4)\).
  • 1) Центрируйте данные.
  • 2) Вычислите ковариационную матрицу \(S\).
  • 3) Найдите собственные значения/векторы.
  • 4) Определите первую главную компоненту.

Решение (вручную).

  • \(\bar{x} = (2,3)\). Центрированные точки: \((-1,-1), (0,0), (1,1)\).
  • \(\tilde{X} = \begin{pmatrix}-1 & -1\\ 0 & 0\\ 1 & 1\end{pmatrix}\).
  • \(S = \frac{1}{2}\tilde{X}^\top \tilde{X} = \begin{pmatrix}1 & 1\\ 1 & 1\end{pmatrix}.\)
  • Собственные значения: \(\lambda_1 = 2, \lambda_2 = 0\), векторы: \(w_1 \propto (1,1)\), \(w_2 \propto (1,-1)\).
  • Первая PC — направление \((1,1)\), вдоль которого лежат точки.

Explained variance: формулы

  • Собственные значения: \(\lambda_1 \ge \dots \ge \lambda_p\).
  • Общая дисперсия: \[ \text{TotalVar} = \sum_{i=1}^p \lambda_i. \]
  • Доля объяснённой дисперсии: \[ \text{EVR}_k = \frac{\sum_{i=1}^k \lambda_i}{\sum_{i=1}^p \lambda_i}. \]
  • Scree‑plot — график \(\lambda_i\) или \(\text{EVR}_k\) по \(i\), поиск «колена» кривой.

Практика 3: scree‑plot (Iris)


import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris

iris = load_iris()
X = iris["data"]

pca_full = PCA(n_components=X.shape[1])
pca_full.fit(X)

eigvals = pca_full.explained_variance_
evr = pca_full.explained_variance_ratio_
cum_evr = evr.cumsum()

fig, ax = plt.subplots(1, 2, figsize=(10, 4))

ax[0].plot(range(1, len(eigvals)+1), eigvals, marker='o')
ax[0].set_xlabel('Компонента')
ax[0].set_ylabel('Собственное значение')
ax[0].set_title('Scree-plot')

ax[1].plot(range(1, len(cum_evr)+1), cum_evr, marker='o')
ax[1].set_xlabel('k')
ax[1].set_ylabel('Кумулятивная доля дисперсии')
ax[1].set_title('Cumulative EVR')

plt.tight_layout()
plt.show()

Практика 3: biplot (Iris)


import numpy as np
import matplotlib.pyplot as plt
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris

iris = load_iris()
X = iris["data"]
y = iris["target"]
features = iris["feature_names"]

Xc = X - X.mean(axis=0, keepdims=True)
pca = PCA(n_components=2)
Z = pca.fit_transform(Xc)
W = pca.components_.T

plt.figure(figsize=(7, 7))
plt.scatter(Z[:, 0], Z[:, 1], c=y, cmap='viridis', alpha=0.7)

for i, f in enumerate(features):
    plt.arrow(0, 0, W[i, 0]*3, W[i, 1]*3, color='r', head_width=0.05)
    plt.text(W[i, 0]*3.2, W[i, 1]*3.2, f, color='r')

plt.xlabel("PC1")
plt.ylabel("PC2")
plt.axhline(0, color='grey', linewidth=0.5)
plt.axvline(0, color='grey', linewidth=0.5)
plt.title("PCA Biplot (Iris)")
plt.grid(True, alpha=0.2)
plt.show()

Практика 4: PCA + k‑means (Iris)


from sklearn.datasets import load_iris
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

iris = load_iris()
X = iris["data"]
y = iris["target"]

pca = PCA(n_components=2)
Z = pca.fit_transform(X)

kmeans = KMeans(n_clusters=3, random_state=42, n_init="auto")
labels = kmeans.fit_predict(Z)

plt.figure(figsize=(6, 6))
plt.scatter(Z[:, 0], Z[:, 1], c=labels, cmap='viridis', alpha=0.8)
plt.xlabel("PC1")
plt.ylabel("PC2")
plt.title("k-means в пространстве PC1–PC2 (Iris)")
plt.show()
