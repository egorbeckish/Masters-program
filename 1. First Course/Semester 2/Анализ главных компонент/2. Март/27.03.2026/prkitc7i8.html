1. Факторный анализ (FA) – основы

Факторный анализ – метод снижения размерности, который предполагает, что наблюдаемые переменные линейно зависят от латентных (скрытых) факторов плюс специфическая ошибка. В отличие от PCA, который просто ищет направления максимальной дисперсии, FA стремится объяснить корреляции между переменными через общие факторы.

\[ X = \Lambda F + \varepsilon \]

где \(X\) – матрица наблюдаемых переменных, \(\Lambda\) – матрица факторных нагрузок, \(F\) – факторы, \(\varepsilon\) – уникальные ошибки (специфические дисперсии).

Ключевые понятия:

  • Факторная нагрузка (loading) – корреляция между переменной и фактором. Значения >0.5 считаются значимыми.
  • Общность (communality) – доля дисперсии переменной, объясняемая общими факторами: \(h_i^2 = \sum_{j=1}^k \lambda_{ij}^2\).
  • Специфическая дисперсия (uniqueness) – \(1 - h_i^2\).

Для проверки применимости FA используют:

  • KMO (Kaiser-Meyer-Olkin) – мера адекватности выборки. Значения >0.6 приемлемы, >0.8 хорошие.
  • Тест Бартлетта (Bartlett’s test of sphericity) – проверяет, является ли корреляционная матрица единичной (нулевая гипотеза: переменные некоррелированы). p-value < 0.05 означает, что FA имеет смысл.

2. Вращение факторов: Varimax

После выделения начальных факторов (обычно методом главных факторов или максимального правдоподобия) факторные нагрузки часто трудно интерпретировать. Вращение (rotation) упрощает структуру, максимизируя дисперсию квадратов нагрузок внутри каждого фактора. Varimax – ортогональное вращение, которое делает нагрузки либо высокими, либо низкими, облегчая интерпретацию.

Математически Varimax максимизирует: \( \sum_{j=1}^k \left[ \frac{1}{p} \sum_{i=1}^p (l_{ij}^2)^2 - \left( \frac{1}{p} \sum_{i=1}^p l_{ij}^2 \right)^2 \right] \), где \(l_{ij}\) – факторные нагрузки, \(p\) – число переменных.

После вращения факторы остаются некоррелированными (ортогональное вращение). Существуют и косоугольные вращения (Promax), если предполагается корреляция факторов.

3. Факторный анализ на данных Mall Customers

Используем библиотеку factor_analyzer (установка: pip install factor-analyzer). Набор данных содержит возраст, годовой доход, оценку трат (Spending Score). Мы применим FA для выявления скрытых факторов.

import pandas as pd
import numpy as np
from sklearn.preprocessing import StandardScaler
from factor_analyzer import FactorAnalyzer
from factor_analyzer.factor_analyzer import calculate_kmo, calculate_bartlett_sphericity

# Загрузка данных
df = pd.read_csv('Mall_Customers.csv')
X = df[['Age', 'Annual Income (k$)', 'Spending Score (1-100)']]
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# KMO и тест Бартлетта
kmo_all, kmo_model = calculate_kmo(X_scaled)
chi2, p_value = calculate_bartlett_sphericity(X_scaled)
print(f'KMO = {kmo_model:.3f}')
print(f'Bartlett test: chi2={chi2:.2f}, p-value={p_value:.4e}')

# Факторный анализ (метод главных факторов, вращение varimax)
fa = FactorAnalyzer(n_factors=2, rotation='varimax', method='principal')
fa.fit(X_scaled)

# Нагрузки
loadings = fa.loadings_
print("Факторные нагрузки:\n", loadings)
print("Общности:\n", fa.get_communalities())
print("Доля объяснённой дисперсии факторами:", fa.get_factor_variance())

Интерпретация: KMO > 0.6 и p-value < 0.05 подтверждают применимость FA. Факторные нагрузки после Varimax: возможно, фактор 1 объединяет доход и оценку трат, фактор 2 – возраст. Общности показывают, какая часть дисперсии каждой переменной объяснена двумя факторами.

📊 Результат на Mall Customers (пример):
KMO = 0.71, Bartlett p < 0.001; после вращения Varimax нагрузки: доход 0.85, оценка 0.78 на факторе 1; возраст 0.92 на факторе 2.

4. Сравнение FA и PCA на наборе Iris

На данных Iris (4 признака: длина и ширина чашелистика и лепестка) проведём PCA и FA (с вращением Varimax) и сравним нагрузки/компоненты.

from sklearn.datasets import load_iris
from sklearn.decomposition import PCA
from factor_analyzer import FactorAnalyzer
import matplotlib.pyplot as plt

iris = load_iris()
X = iris.data
scaler = StandardScaler()
X_std = scaler.fit_transform(X)

# PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_std)
print("PCA loadings (компоненты x признаки):\n", pca.components_.T)

# FA (2 фактора, varimax)
fa = FactorAnalyzer(n_factors=2, rotation='varimax', method='principal')
fa.fit(X_std)
print("FA loadings после Varimax:\n", fa.loadings_)

# Визуализация (скетч кода)
plt.figure(figsize=(12,5))
plt.subplot(1,2,1)
plt.scatter(X_pca[:,0], X_pca[:,1], c=iris.target, cmap='viridis')
plt.title('PCA: первые две компоненты')
plt.xlabel('PC1'); plt.ylabel('PC2')
plt.subplot(1,2,2)
X_fa = fa.transform(X_std)
plt.scatter(X_fa[:,0], X_fa[:,1], c=iris.target, cmap='viridis')
plt.title('FA: факторные координаты')
plt.xlabel('Factor 1'); plt.ylabel('Factor 2')
plt.show()
📊 На Iris PCA и FA дают разное геометрическое представление: PCA ориентирует оси по максимуму дисперсии, FA – по общим факторам. Нагрузки после Varimax часто делают факторы более интерпретируемыми (например, фактор 1 – размеры лепестков, фактор 2 – размеры чашелистиков).
Главное отличие: PCA – это преобразование данных, FA – модель, объясняющая ковариацию. В FA мы предполагаем наличие латентных факторов, PCA просто ищет направления.

📘 Задание 1: Вращение вручную

Возьмите простую матрицу нагрузок 2×2: \[ \Lambda = \begin{bmatrix} 0.8 & 0.3 \\ 0.4 & 0.7 \end{bmatrix} \] Примените ортогональное вращение Varimax (аналитически или используя формулу для двух факторов) и получите повернутую матрицу. Объясните, как изменилась интерпретация.

Подсказка: Для двух факторов вращение – это поворот на угол θ. Varimax максимизирует сумму дисперсий квадратов нагрузок.

✅ Пример решения 
import numpy as np
from scipy.optimize import minimize
L = np.array([[0.8, 0.3], [0.4, 0.7]])
def varimax_2d(L, max_iter=100):
    # ортогональное вращение для двух факторов
    def criterion(theta):
        c, s = np.cos(theta), np.sin(theta)
        T = np.array([[c, -s], [s, c]])
        Lrot = L @ T
        # критерий Varimax: максимизация суммы дисперсий квадратов нагрузок
        v = np.sum((Lrot**2)**2) - np.sum(Lrot**2)**2 / Lrot.shape[0]
        return -v
    res = minimize(criterion, 0)
    theta = res.x[0]
    c, s = np.cos(theta), np.sin(theta)
    T = np.array([[c, -s], [s, c]])
    return L @ T
print(varimax_2d(L))

Результат: после вращения нагрузки становятся более "поляризованными" (например, первый фактор сильнее связан с первой переменной, второй – со второй).
